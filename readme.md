# How to
```
Is He Baiting For Frags? - The Age Old Question
Usage of IsHeBaitingForFrags.exe:
  -demofile string
        The demo file to parse (default "test.dem")
  -steamId int
        SteamId64 of player to Watch (default 76561198128945703)
  -topBaiters
        Analyze every player
```
